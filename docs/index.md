# Bem vindo(a) a documentação do pacote Quadros de Saúde!

`quadrosdesaude` é um pacote Python desenhado para facilitar o processamento de dados de saúde pública do Brasil, com foco inicial nos ficheiros do DATASUS.

## Instalação

Você pode instalar o pacote diretamente a partir do PyPI usando o pip:

```bash
pip install quadrosdesaude
```
